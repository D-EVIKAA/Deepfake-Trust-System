Deepfake Trust & Attribution System

A system that evaluates the trustworthiness of audio and video instead of simply detecting fake media.

Features

- Media upload (mp4, avi, wav, mp3)
- Metadata analysis
- Audio signal analysis
- Video frame analysis
- Compression detection
- Trust score calculation
- Risk level classification
- Explainable findings
- Watermark / provenance system

Tech Stack

- Python
- Streamlit
- OpenCV
- Librosa
- NumPy
- Hugging Face ML models

 How it Works

1. User uploads media.
2. The system analyzes metadata, audio signals, and video frames.
3. Machine learning models evaluate AI-generated patterns.
4. A trust score and risk level are calculated.
5. Results are displayed with explanations and watermark verification.
